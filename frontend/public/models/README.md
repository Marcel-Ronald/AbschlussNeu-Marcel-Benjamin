# 3D Hai-Modelle

## Kostenlose Hai-Modelle herunterladen

Du kannst kostenlose, realistische Hai-3D-Modelle von diesen Seiten herunterladen:

### 1. Sketchfab (empfohlen)

- https://sketchfab.com/search?q=shark&type=models&features=downloadable&sort_by=-relevance
- Filter: "Downloadable" aktivieren
- Format: GLB oder GLTF wählen
- Lizenz: CC BY oder CC0 (kostenlos nutzbar)

### 2. Free3D

- https://free3d.com/3d-models/shark
- Suche nach "shark"
- Kostenlose Modelle herunterladen

### 3. CGTrader

- https://www.cgtrader.com/free-3d-models/animals/fish/shark
- Filter: "Free" auswählen

## Modelle einbinden

1. Lade ein Hai-Modell herunter (Format: .glb oder .gltf)
2. Benenne es um zu:
   - `great_white_shark.glb` (Weißer Hai)
   - `whale_shark.glb` (Walhai)
   - `hammerhead_shark.glb` (Hammerhai)
3. Speichere es in diesem Ordner (`public/models/`)
4. Die App lädt die Modelle automatisch
